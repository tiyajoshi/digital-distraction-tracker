# Digital Distraction Tracker 📊

A command-line productivity tool that helps students track their focus and distraction habits in real time.

## What it does

- Tracks time spent on tasks and labels them as **productive** or **distracting**
- Calculates a **productivity score** at the end of each session
- Saves your history so you can review past sessions
- Gives feedback based on your score to help you stay focused

## How to run it

Make sure you have Python 3 installed, then:

```
python3 tracker.py
```

## Features

- Track unlimited tasks per session
- Data is saved automatically — nothing is lost when you close the app
- View your full task history with timestamps
- Get a productivity score and personalized feedback

## How to use it

1. Run the program
2. Choose option `1` to start tracking a task
3. Enter the task name and whether it is productive or distracting
4. Press Enter to start the timer, press Enter again to stop
5. Choose option `2` to see your productivity summary
6. Choose option `3` to view your full history

## Built with

- Python 3
- JSON for data storage
- datetime and time modules
